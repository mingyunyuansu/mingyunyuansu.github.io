A Byte of Python clear读完

Sep 24, 2016 | 杂记

今天没有学日语。。。

但是我可以骄傲地说我终于掌握了Python，因为我今天背上电脑与电源去图书馆坐了一天，把一本A Byte of Python 读完了，而且是精读，写了半本笔记本。。。

之前断断续续了解了一些Python的内容，因为简单，所以太轻视了，今天本来想学flask，结果发现一脸懵逼。最后一狠心找来这本号称最好的教材一口气读完。

只能说 畅快淋漓 反正这书就是对我胃口，每个知识点一段定义，一段例子，一段讲解， 没有一句废话

同时适当的带一些小细节，又不至于让人太钻牛角尖。。。同时还有针对各种语言的程序员的tips，看得我感动啊，真的做到了新手老手因材施教。。。反正怎么吹都不过分啦。

我的读书法则定理.append(‘一本书读80%只能获得20%的精髓，读完100%才能获得100%’)

另外Python真的可以作为现代新一代编程语言的代表，真的是 简易 而不 简单 ，同时很 优雅。

唉，目前为止，大学最大的收获应该在于：如果你愿意，真的可以一天学完一本书（source：每学期期末考最后一周）

总之希望接下来每天都能这样过啦，最大的敌人不是dota不是动漫，而是会不知不觉度过一个两个三个小时却不知道自己干了什么。
