Python创建多维数组的一个大坑

Dec 23, 2019 | 技术

Python是一门神奇的语言。它几乎帮你省去了所有编程语言自身的麻烦特性，使得人类可以用最纯粹的思维和语言来编写代码，而不用在思考算法的时候同时思考“在某一门语言”上如何实现（比如在C下单纯的想使用链表结构，那写指针就要费一会儿功夫）。

本来应该是这样。

但是事实是，因为Python省去了太多本来你应该考虑的细节，比如变量类型，比如内存空间，从而导致有时会出现莫名其妙的错误而不自知。
