942.DI String Match

Given a string S that only contains “I” (increase) or “D” (decrease), let N = S.length.

Return any permutation A of [0, 1, …, N] such that for all i = 0, …, N-1:

  • If S[i] == “I”, then A[i] < A[i+1]
  • If S[i] == “D”, then A[i] > A[i+1]

挺简单的，想了几分钟，又写了几分钟就OK了。我的想法是，首先建一个0到N的有序数组作为初始的A，然后顺着过一遍S，根据碰到的’D’和’I’，判断对应位置的A[i]和A[i+1]是否符合要求，不符合就swap(A[i], A[i+1])。因为A比S多一位，所以没有边界问题。但是直觉就是肯定会出现这种情况：swap当前的数字影响了前面已经调整好了的数字。不过没关系，仔细一想就会发现这种情况只会出现在连续的’I’段或者连续的’D’段中，比如对于[0 1 2]和”DD”，对第一个’D’执行swap，得到[1 0 2]，对第二个’D’再执行swap得到[1 2 0]，这时就会发现第一个’D’受到了影响，变得不符合条件。那么很自然地想到，每次swap时确保在当前位置之前的连续的’D’段（或’I’段）都符合条件就行了，代码如下：

毕业&旅行&分别

在我刚从武夷山回来后，我们宿舍还是决定一起去一趟毕业旅行，时间定在毕业典礼之后。商讨的过程非常有我们宿舍的风格：随机的商讨之后由某个人随口的提议决定了去苏杭一带，然后在经过一段不知为何要有但就是要有的拖延之后的某一天，由我在颓废之余随缘地购买了火车票，定下了为期一周的旅程并计划了每一天的行程。宿舍四人中厦门和泉州的两个是没有什么旅游经验的，态度也较为被动；而我又懒又没动力，所以主要计划人还是北京那个，有点过意不去。而且在我知道我们的计划是一星期后，直觉就告诉我这是不太可能的，不管是我的体力方面，还是以我对另外两人的了解。但是不管怎么样，先决定了总是好的。

之后就是毕业典礼了。典礼在建南大会堂，早上很早就要起床，大雨，路上又发现忘了带冷餐券，折腾一番后到达会场时下半身已经变得被水泡过一样了，体验非常糟糕。我记得当年开学典礼也来过这里，没有空调热的要死，现在毕业典礼则是在水中进行。开场后一群领导说了一堆不知所云的话，我努力地暗示自己这是一个充满仪式感的场面，但还是失败告终，被无聊淹没。不过领导中有纪荣嵘，我也不记得什么时候听过他的什么故事，反正听到这个名字的时候总是浮现出一丝憧憬的感觉，可能我曾经是憧憬过那样的人的吧。我顺便听到了陈贤鹏没有来参加毕业典礼，他因为延毕了而不想来。他平时总是一副乐呵呵的样子，让人唏嘘；几天前也看到班群里有撑不下去了求助心理医生的同学，可能觉得生活索然无味的不止我们宿舍吧。我们整个班确实直到毕业都挺沉闷的，对比朋友圈里那些欢快的毕业照，也不知道是什么问题。典礼最后就是飞快地上台拍个照拿一张毕业典礼纪念纸就被轰出去了，甚至都没发毕业证，果然毫无仪式感。冷餐会结束后，就意味着绝大多数相处了四年的同学再也不会见面了，但是其中我怕是有四分之三没说过话，所以就这样咯，没人会惋惜。晚上我还是联系了喜欢了三年的女生，她因为导师安排去深圳实习，毕业照跟毕业典礼都没来。我没有告白，只是用一种比较暧昧的口气尬聊了一会儿，从她的反应我想她应该是明白的，毕竟这三年我对她有各种忽冷忽热的不自然表现。气氛有点尴尬，我就先告别了，这就算结束了吧？整个大学，我都在做这种不干不脆的事情，学习也好，恋爱也好，生活的信心究竟去了哪里呢，没了学习成绩的我原来什么都不剩吗？之后是清理宿舍，真的有够辛苦，对以后要住的房子该是什么样有了更坚定的打算。

注意上面是七月份写的开头，接下来的内容写于2018年11月4日，据旅行时间（6月26日）已经过去了四个月有余。这几个月我的生活不尽人意，令人疲于应对，所以旅行的许多细节我可能想不起来了。不过我想能够被一直铭记下来的那些事情才是真正重要的东西吧（诡辩）。

时间来到旅行出发当天，这真是一趟充满戏剧化的出行。。。首先，出发的前一晚上我们就已经把宿舍门钥匙和空调遥控器交到了保卫处，所以第二天早上留了一堆实在收不动了的垃圾（保洁阿姨对不起），和北京的舍友二人就带着行李，背着毕业证匆忙地（同时也是永远地）离开了这个宿舍。按照计划是，我们四人中，我和北京的从宿舍出发，厦门的从家出发，泉州的从自己租的房子出发，最后在火车站集合，坐上十点左右的火车一起出发。但是我刚起床就看到厦门那个同学昨晚一点多给睡梦中的我发了微信，按照以往经验，这个逼会主动联系别人八成是又有麻烦事了。果不其然，他的出国申请资料出了啥问题，总之他今天需要去本部学生办事大厅打印学位和毕业证明然后再送去中介那里才行，并且那里八点（左右）才开门。说起来，我们两个以后也是要出国，这些证明我们也需要啊。。。在公交车上不停地计算时间，似乎是来不及了，又打电话给他，他说他已经在排队等着打印了，让我们先走吧，咱们杭州见，就匆忙挂了电话，干哦！当时我正好在看比宇宙更遥远的地方，里面有一个很类似的情节，我很受触动，再说毕业旅行四个舍友都不能一起出发，在杭州再拼凑在一起，那是旅行团吗？从大一开始这样这破宿舍就总是这样，难道这点事都办不好？于是我当时就决定了去打印处找他。去到打印处处后，果不其然，这是离校第一天，又是英文学位证明开放打印的第一天，排起的长龙直接到了房间外。我们找到了他，并且说明我也要打印，所以过来了。这时他有点生气，说难道我认为这次离校以后在厦门给的他不会再帮我们打印？我当然是觉得可以让他之后再帮忙打印的，但是由他这样说出来还是有点感动。我表明我们四个人一定要一起出发，然后把火车票往后改签了一个多小时（一次改签居然免费），时间就充裕多了。总之这一来一回终于有点舍友的人情味了。

之后我们三人就坐上出租车去找他的留学中介了。车上广播节目在谈今年厦门高考700分以上有十多个，我突然想起之前看到今年河北高考700分以上有100多个，不知为何这段镜头是整个旅行印象最深的几个片段之一。到了之后，北京那个说正好他也要找中介谈，他们两个就一起进了大楼，感情你们俩是一家中介。。。于是我坐在门口等。那一天天很蓝，视野也是一层蓝氲，大楼正好在鹭江道，隔海就是鼓浪屿，海风吹拂，我坐在路边看着晨光下人来人往车水马龙，旁边是三个行李箱。前途一片黯淡，此刻的我却反而感受到了一丝青春的气息。这一幕我觉得很有戏剧感，大概就是公路片的感觉？特意拍照留念了。


然后去到火车站，泉州的老哥又果不其然地掉了链子，不知道他到了哪里，总之他找不到火车站的进站口，费了些时间，最后终于在还剩十分钟不到时240全员聚集，可喜可贺。然后就是按部就班地坐火车。不过似乎是因为改签的关系，我们原本坐在一起的座位变成了有一个人要单，最终厦门的那个坐了那个位子，这个逼一个人也没什么不满，居然拿出一本近代中国史津津有味地读了起来，旁边的乘客都看呆了，这个宿舍果然全员怪人。订了酒店后一路发着呆吃着零食，等到车窗外的风景变得“江南”了起来，就知道差不多到了。我很难形容这是种什么感觉，就是有青山有平地有水，但是又跟云南那种有力的山水不同，土地都被一层柔柔的油油的绿草覆盖了，山峦在清冷的烟雾中若隐若现，和天上的云雾分不清，加上到处都是河流池塘，水乡这个名字还真是形象。我还看到周边的村落建在阡陌交错的水道之中，就是威尼斯的那种感觉，吃惊的是家家户户门口的小河里居然像我们停摩托一样停了一张张水上摩托，emmmm这果然很江南。到站下车，我前面的小朋友在座位上跳，一个男人，大概是他爸，手扶着他，眼睛却目不转睛一点不遮掩地盯着他前面那个打扮风骚的女人，直到小朋友从座位上摔下来才发现手已经放开了，真实。我可能需要学习一下正常的男人都是什么样的。

下了火车，即使太阳落山了，热浪依旧扑面而来。打算坐地铁去酒店，发现这里地铁也大力推行支付宝业务，还有银联闪付，四人研究半天，不知道是机器问题还是什么，反正搞不定，最后还是排队走人工售票。买完之后中还看到一个男的正大光明的插队了，汗，还以为这里素质会很高。仔细一看其实地铁站也有点破旧，其实上限很高下限也很低才是真正的大城市吧。上了地铁，一路上我发现这里的妹子好像真的颜值略高，皮肤白皙细腻，长相温婉，难道真是传说中的下有苏杭。最后到了酒店，发现好破旧，湿气还重，不过是在南宋御街，本身就在景点里，离地铁也近。但是明明订的两个一样的双人间，却有一间要小一点，而且空调还吵，下去找前台学我妈的样子气势汹汹要求更换，就给换了。然后就到了喜闻乐见的分房间的时候，他们三个都异口同声要跟我一间，也不知道是我受欢迎还是北京那个的“特殊”原因，或者二者兼有，总之最后考虑到特殊情况，我还是和北京那个一间了。北京那个向我表示这个酒店档次太低，其实订酒店时我就注意到了厦门泉州两位反而认为这个价位有点太高，就是为了防止这种情况才选了这么个折中选项再三确认才订下的，都给我闭嘴。这个宿舍的一大问题就是个个有话不说，偏偏个个都心思很多，盔甲裹得老厚，真的烦，墨迹的人有我一个就够烦了，现在这个宿舍有四个。

第二天就正式开始走景点了。原谅我，隔了四个月，我整个行程的头尾都记得还算清晰，唯独中间的旅行的过程变得模糊了起来。。。不知道是因为这里“厦门对不起”的爆炸温度，还是缩水的景点，或者我就是不太喜欢串景点大脑帮我自动模糊了。只记得第一天（其实似乎不是第一天）去的是西溪湿地。大概就是一个普通的湿地公园，而且在我眼中真的很水。。。植被一般般，自然景观一般般，就是普通的山林里的感觉，坐了游览船在几个不太懂的景点间转移了一下，途经一个区域甚至在介绍这里的原住民的民俗生活，摆了点桌椅板凳生活用品模型，最后的目的是为了卖以这里民俗为卖点的手工丝质特产服装，就是各旅游景点的常见套路。可是我的天，你不是个湿地公园吗，是要闹哪样。要命的是，这路难走还很长，连游览船上都没空调，路上两个老外已经累得脸红脖子粗（Red neck)，露出完全茫然的表情，我不禁有点愧疚。中午在一个类似丽江古城的商业区（还特么没开发完）找了一家冷饮店休息，看来大家都是碰到空调走不动路，于是四个人拿出了卡卡城在丽江的风景区里开始开心地玩了起来。说起这个卡卡城，大概是大二大三的时候北京那个就买来希望大家能一起玩的，却一直因为（主要是我）嫌麻烦，导致一直寂寞地被放置，还算最后发挥了用处，大家玩的不亦乐乎，后来甚至可以说是杭州之行最重要的事宜之一（其实我记得学习如何玩卡卡城是某天晚上在一家肯德基里，所以我才说西溪大概不是第一天去的）。最后下午了我们去了西溪的最后一个景点高庄，似乎是从前一个贵族家的房子，确实是建的古色古香，很有江南味，我身位一个中国人，却感觉到和这种正统的中原文化略有格格不入，房子的许多设计细节，假山，青石板，题字，门阖，中堂，屏风，像电视剧一样，云南确实是没有这种文化氛围的。这个高庄可以说是西溪的唯一亮点了。之后就坐车回了酒店。

然后我就连哪天是哪天都乱不清楚了。总之，我们在西湖花了大量时间，因为西湖其实是一个风景区的总称，不止西湖本身。先去坐船游了一圈西湖，售船票的老爷爷并不会看学生证上“已毕业”的印章，于是成功按学生票薅了一把羊毛。其实西湖水灰灰的，船上又热，水汽蒸蒸，但是乘船游西湖，眺望雷峰塔这个事本身的仪式感很重要，还好。之后就徒步绕湖游览了，一路上我发现基本只要是块石头是面墙是个亭子都会有人题字，然后变成景点，还不乏大家，毛泽东康有为什么的，这种被文化和历史包围的感觉也是只有这种地方才能感受到，也不知道在这种城市长大的人他们的道德观是什么样的。然后来到了一元人民币纸币上那个著名的西湖风景图，就是有三座不知是什么东西的墩墩和一座桥的那个地方，纷纷照了相才发现似乎角度不对，是湖对岸才能拍出人民币那副画。说起来我上次来到这里时才四岁，也拍了这个图，不知道景色变了多少。再走着走着感觉真的要不行了，实在太热了，掏出手机一看体感温度45度尼玛，纷纷决定撤退了。一起去南宋御街上一家号称杭帮菜馆的餐馆里吃饭。我个人觉得这种地方又大张旗鼓表示自己是杭帮菜的菜馆多半都很水，不过试试也无妨，点了些特色菜，西湖醋鱼就是一条白灼鱼，然后倒上一层浓浓的糖醋，这味道有点欣赏不来。然后是大名鼎鼎的东坡肉，一人点了一份，上来了是一个小碗，装一坨超级带皮大猪肉，一口下去这油我是有点hold不住，不愧是苏大胡子。。。但是其实味道还行，而且做得已经没有那么腻了（虽然还是hold不住）。这顿饭最受欢迎的猪油渣煮魔芋白菜汤，味道挺好，不过北京那个认为这菜没啥亮点，确实是这样，但是好吃也很关键啊。最后结账意外的很便宜，至少不是宰客，味道不说好吃，也还不错。

依然是记不得哪天，我们还去了雷峰塔，雷锋塔建设倒是挺不错的，毕竟真正的雷峰塔已经倒了，新的是2002年在老的遗址上重建的，内部还能看到一些残垣断壁。上到塔顶居然还挺高的，放到现代也算是高楼了，对古人来说应该算是相当震撼的。在西湖周围绕了一绕，去景区肯德基吃了点东西，然后在门口一家特产小卖部问了一下支付宝的电子交通卡怎么用，就决定去灵隐寺了。话说，因为去要坐公交，我们不会弄这个交通卡，然后小卖部的妹子虽然懒懒散散的，说话淡淡的，手上却热心的帮我们在手机上注册绑定好了交通卡，这种傲娇（？）对我很受用诶，我感觉啥都不买有点过意不去，一直唠叨是不是买把扇子，出店来被剩下三人嘲笑怕不是因为她长得好看，咳，也有可能吧，真是残酷的现实。去灵隐寺的路上，还路过了一个什么庙，不算很大，本来想进去转两圈，但是一问要门票，我看泉州厦门舍友就满脸不乐意，而且还唯唯诺诺不说出来，我真的一看气不打一处来，你都来杭州旅游了却连几十块的门票都不愿买，不愿意就别去，于是被我拉着出来直奔灵隐寺。灵隐寺，完全没了解，只是因为这个名字太酷了，所以想来了，也不排除我在哪里听到过。到了灵隐寺，不得不说，确实是器宇恢弘，整个寺顺着山建，一大间一大间的大殿，周围鳞次栉比的小殿，层层递进。大殿之中，各色佛神和魑魅魍魉的雕像各具特色，浮雕雄伟瑰丽，相当惊艳。各种巨型装饰和题字，当大这个属性被凸显的时候，就只剩下令人肃然起敬的感觉了。就算是我这种反宗教者也不得不承认这些东西的价值。至于小殿嘛。。。就跟博物馆差不多了，相比大殿，我觉得里面的空调也非常夺人心魄！然后是飞来峰，我至今也不知道飞来峰这个东西究竟是指整个风景区还是有什么玄机，反正进了大门就不知道该去哪里了，我们就信步探险，还真找到了一些石洞和佛像雕刻，非常有年代感和年久失修的感觉，感觉有点东西。接着顺着山路看着百度地图向上爬，打算去飞来峰。飞来峰某种意义上是这个旅程最令人“印象深刻”的景点了，高温之下随着难走的山路摸索上来，到达的飞来峰却是一个“小山头”。。。既没有登高望远的感觉，因为虽然路难走单整个山海拔都不高，山顶也没有任何有意思的景观。最气人的是，如果不是拿一块石头，用微软幼圆的Word字体写上“飞来峰顶”放在这里，你甚至想不到这是飞来峰，周围游人无不懊恼。也不知道哪里出了问题，反正王安石我只想给你一拳。

就景点部分，我只记得那么多了，应该没有什么遗漏了。这次是毕业旅行，我觉得宿舍几个人一起走就是全部了（虽然好多景点确实也没什么意思。。。）。也还有些有意思的小事，在南宋御街吃到了人生第一次DQ，在肯德基狂吃肥宅快乐桶，并在四周看nerd的表情中，四个人争强好胜打卡卡城，感觉我们就像生活大爆炸的几个人。介绍了生煎包给他们之后，天天跑来咬不得高祖生煎这里把菜吃了个遍（顺便一提我还把手机落这里了，表情冷淡的小姐姐替我收着并确认我能开锁才还给我）。在酒店里的每个晚上都和北京舍友谈了许多，以前的矛盾，真实的想法，算得上是推心置腹了，相信另一个房间的两个人也是如此。这些一幅幅场景刻在脑海中，某种程度上比景点更重要吧。在离开的前一天下午，我心情很奇怪，一方面想着旅程终于结束了，虚无的大学生活也终于结束了，另一方面却又对旧的人旧的时间非常不舍。趁着天色没暗，让推销的小哥帮忙拍一张合照，没想到背后的牌坊的题字，刚好就是我校校训“止于至善”，这种巧合可以说非常完美了。我当然不相信什么天意这一说，但是世上的偶然和必然在加入了观测者的期待之后就出现了天意，不失是浪漫的。

最后来到了分别的时候。因为之前一直没有商量到底什么时候离开，我有点舍不得，迟迟没订飞机票，没想到后面一问这帮逼早就订完了，还一个比一个早。我赶紧订机票，机票很紧张，最后票价1300，超疼，好气。泉州厦门二人订了早上十点的火车，毕竟同一个方向可以作伴，北京舍友订了之后的高铁，我最后，下午萧山机场的飞机。这天下着雨，一早从酒店退房出来，无话地走在去地铁的路上。先送最早的两个人，目送他们下了扶梯，没有眼泪没有拥抱也没有太多台词，就是普通的拜拜而已，就像每一次寒暑假回家，甚至就像每天有人回宿舍有人去食堂的一句“走了”而已。我从小到大经历过不少这样的离别，和发小的也好，和同学的也好，都太平淡了，没有哪次能够处理好。但是我看到他们一直有话想说欲言又止的表情，想必我也是一个表情，大概就这样也不错吧，你懂我我懂你，也不是能随便求得来的。就剩我跟北京舍友了，要我给拍张照，拍完紧接着他的地铁先来，这次就更快了，更来不及就走了。感觉瞬间就剩我一个了，这种感觉寂寞但是决绝，我知道接下来的地铁公交飞机都将送我前往新的道路。我坐地铁到公交车站买票去萧山机场，一起坐公交的有一个MTF，这也是某种缘分？杭州的高速路干净而清冷，加上下着雨，车内也相当冷清，可以说很应景了。我掏出耳机听着歌，脑海中想了过去，想了未来，也默默再次缅怀了一下240。不过没有持续太久，百度地图显示一小时的路程其实只开了十九分钟。。。按部就班坐上飞机回到昆明，见到了接机的爸妈，那是七月的第一天，到今天我感觉真的很难熬，像过了很久，不过那是另一个故事了。

对于班级呀学校啊这种的分别，我一直都是后知后觉，直到四个月后的今天，我才突然有一种难以抑制的怀念和伤感，回想起宿舍中的一幕幕，又想到这些光景一去不复返了，就无比遗憾。其实人真的很奇怪，尽管传统文学文化都会着重渲染离别的哀思，但是其实放在今天的标准来看就算都不在一个国度了，只要你还在地球上，想见面不都是分分钟的事？但是我们不会去那么做（特别针对这个宿舍这种性格来说），地理上的便捷性也反直觉地不会缓解离别的悲伤，这种悲伤其实来自于我们都明白，不管多不舍，我们今后的人生不会再有焦点了。

游武夷山

6月16号到6月17号，爸妈利用端午节假期，我们全家一起去武夷山玩了两天，然后18号他们来厦门帮我带部分行李回去，也算作我们这一家结束和厦门的缘分。一路上发生了一些有意义的事情，随便记记。

6月16号早6点半，我起来福至心灵把要签名的报到证复印件签了名，便出发上路。从公寓出发去厦门北站本来挺远的，30多公里，但是一趟公交转地铁就直达了，有了地铁（虽然只有一条线）比以前方便，时间计算也舒服很多。高铁一路无事，不表。

Floyd's Tortoise and Hare

之前提过，用速率不同的两个指针遍历一个链表，可以在O(N)时间内判断出该链表是否有环（这是一个很经典的问题，我的舍友面试就遇到然后不幸挂了）。今天机缘巧合，我在看一个找一个数组中的重复元素的问题时发现最佳解法居然还是这种双指针方法（毕竟数组其实就是钉死的链表），也就是说这已经可以抽象成一个有效的算法思想了，果不其然顺藤摸瓜摸到quora上发现这种双指针跑链表的原型叫做Floyd’s Tortoise and Hare算法（那个数组的问题也很神，不过今天就只谈双指针了），这个算法不仅可以确定一个链表是否有环，还能确定环的入口。这听起来不太可能，乍一看由于链表的单向性，当我们处于链表内部时如果不引入外部储存空间则无法看到链表的全局面貌，所以判断条件就无从入手，但是该算法却用简单的数字代换提供了巧妙解法，我非常奇怪为何我们的算法课不教这个算法，是因为太简单了吗？

  • step1 判断环的存在

    这步原理之前其实说过，不过有人跟我说看不懂，所以详细说一说。我们先定义两个指针用于遍历，一个一次走两步（也就是Hare，快指针），一个只走一步（Tortoise，慢指针），两指针同时从链表头出发。显然，由于链表的单向性，如果有环，则遍历会永远持续下去（一直沿环绕）；如果没环，则快指针迟早会到链表尾，所以一旦碰到nullptr即可确定无环。


    可是如果没碰到nullptr，我们不能确定快指针究竟是“还在路上”还是已经沿着环绕了一圈又一圈（因为在链表内部只看得到局部）。好吧，结论就是，如果有环，时间放到无限长，慢指针和快指针肯定都会进入环然后在上面绕啊绕，并且可以证明在绕啊绕的过程中两个指针一定会相遇。


    换言之我们只要在遍历的过程中不断检查两件事，一是快指针是否碰到空，为空则直接表明无环；二是快慢指针是否相等，若相等则表明有环。二者之一必定发生，problem solved。


    但是别人肯定会觉得你说两指针一定会相遇有种钦定的感觉，所以还要说明一下。设一个链表非环部分长度为s，环部分长度为l，某时刻慢指针走了k步，那么快指针走了2k步。当两个指针相遇时，不管他们已经绕了几圈，他们之间的步数差距为肯定为l的倍数，设为n倍，即 (2k-s)-(k-s) = k = nl，k为不断增长的整数，这个式子迟早会成立，也就是说相遇的情况一定存在。这很好理解，两个指针走过了相同的非环部分s，然后在环上快指针比且仅比慢指针多走了nl步，跟没走是一样的，他们会停在同一点。当然你会说这个说法不严谨，因为有可能因为环很长，两指针还没进入环时步差就已经达到l的倍数，式子不适用。但是其实没关系，这式子不必是等价条件，因为时间无限长的情况下两指针一定会纷纷进入环，然后步差为nl的情况就一定会出现，能证明其存在性就可以了。严格的数学证明可以找数学归纳法，反证法，数论什么的，但是我觉得没有这个式子好理解。顺便一提，其实快慢指针速度只要不一样就行，不一定得是两倍，上面的式子依然是成立的，因为n的取值也是所有自然数。

  • step2 确定环入口



    方法是，在第一步找到快慢指针的相遇结点后，让其中一个指针回到链表头，然后两个指针都一次走一步继续遍历，当两指针再次相遇时，该结点即为环的入口。


    为啥？其实很简单，我们不考虑数学也先不考虑两个指针已经绕了几圈，直接设非环部分长s，两指针相遇时的结点相对环起点的偏移量offset是x，如上图。慢指针这时走过的长度至少是s+x，而快指针则至少是2*(s+x)，毕竟任何时候快指针走过的长度都是慢指针的两倍。但是他们相遇了！这说明s+x的长度，恰好是环的长度，才能使得快指针多走s+x的距离依然能与慢指针相遇。那么这时只要让一个指针回到链表头，另一个留在x处，两指针都一次走一步，他们一定会在走过s长度后相遇，相遇的结点即是环入口，简直浑然天成。


    不过一个非常显然的问题就是，上面也说了慢指针走过的距离只不过“至少”是s+x，他有没有多绕几圈我哪知道？其实不妨设慢指针走过的长度为s+x+nl（n=0,1,2,3,.., l为环长度），则快指针就是2(s+x+nl)，结论就变成s+x+nl的长度是环的长度（的倍数），nl本身就是环长度倍数，取模去掉，所以剩下部分结论不变，QED。
