136. Single Number（一个很骚的位操作）

我一直觉得位操作虽然酷炫，但是从来没见过在高级语言中有什么应用场景（因为我比较菜），直到我看到了LeetCode Problem 136.Single Number。
题目是在一个数组中，除了一个数以外其他所有数都有重复，找出没有重复的那个数。想了想似乎没有什么好的办法，单纯迭代太low，哈希表要空间而且我这种土鳖已经全忘了（Python倒是有字典，但是我不想用Python，Python还有求set()呢那不是更简单）。然后一看答案还有个位操作：
原理：

  • a^a = 0
  • 0^a = a
